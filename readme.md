React Starter
=============

Starter Kit for React SPA

- React
- React Router
- Webpack config
- Webpack express middleware
- Babel ES6
- Unit test with Mocha

Setting up
----------

Running app
-----------

In dev:

```
npm install nodemon -g
nodemon
```

In production:

```
npm start
```

Linting
-------

```
make lint
```

Unit tests
----------

```
make test-unit
```
